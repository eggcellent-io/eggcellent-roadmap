Eggcellent is a lightweight tool that turns **YouTube videos and cooking websites**
into clean, structured recipes you can actually cook from.

## Why Eggcellent Exists
Video recipes are inspiring but hard to follow:
- Ingredients are scattered
- Steps are buried in long videos
- Recipe blogs are cluttered with ads

Eggcellent removes the friction.

## What It Does
- Import recipes from YouTube links
- Import recipes from cooking websites
- Generate clean ingredient lists and steps
- Save recipes for later use
